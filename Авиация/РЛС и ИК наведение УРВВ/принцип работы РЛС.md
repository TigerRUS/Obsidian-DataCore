*Главный лепесток (Луч)* - наиболее сильный сигнал служит для поиска и удержания цели и захвата и наведения вооружения.

Если в боковой лепесток что-то будет лететь, то главный лепесток может нечётко видеть цель. От такого эффекта возникают помехи, от чего пущенные радарные ракеты хуже наводятся, и либо УРВВ будет лететь прямо в режиме ПСК, либо ракета уйдёт в цель, засвеченную боковым лепестком.

**Паттерны сканирования**
*Одностороннее растровое сканирование*
Проход радара строится слева сверху, движение происходит слева направо, по окончанию сканирования начинается новый проход с начала.
![[image-16.png]]

*Двустороннее растровое сканирование*
Проход радара строится слева сверху, движение происходит слева направо, по окончанию сканирования начинается проход в обратном направлении.
![[image-17.png]]

Ширина по углу места = количество полос * высота каждой полосы
Максимальный угол подъёма = (ширина по углу места / 2) + вертикальное смещение
Минимальный угол подъёма = (-1 * (ширина по углу места / 2)) + вертикальное смещение
