*succinct data structures* - класс наиболее компактных структур данных.
Канонический пример такой структуры - это представление дерева в виде правильной скобочной последовательности, дерево из n-вершин таким образом представляется с помощью 2n-бит в то время как типичная динамическая реализация требовала бы как два указателя по 64-бит на каждый узел (разумеется можно немного сократить простыми оптимизациями, но даже близко 2 бита не получить).
Фундамент подобных структур - это *rank-select словарь*, представляющий собой битовый вектор и дополнительную структуру для выполнению двух операций ранг и селект. В указанном примере с деревом с помощью ранга и селекта можно сделать базовую навигацию: найти номера потомков/родителей, узнать размер поддерева.
