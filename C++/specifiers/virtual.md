В C++ ключевое слово `virtual` используется для объявления **виртуальных функций** в базовом классе, что позволяет *динамически выбирать* (во время выполнения программы) нужную реализацию метода из производного класса, работая через указатели или ссылки на базовый класс, и является основой для реализации полиморфизма. Оно говорит компилятору: "Эта функция может быть переопределена в наследниках, и мы хотим, чтобы вызов зависел от фактического типа объекта, а не от типа указателя". 

**Объявление:** Вы ставите `virtual` перед функцией в базовом классе:
```c++
class Base {
public:
	virtual void show() { /* ... */ }
};
```

**Переопределение:** В производном классе вы можете переопределить эту функцию (используя `override` для ясности, хотя это не обязательно, но рекомендуется):
```cpp
class Derived : public Base {
public:
	void show() override { /* ... */ }
};
```

 **Вызов:** При работе с указателем/ссылкой на базовый класс, но указывающим на объект производного класса, будет вызвана версия функции из производного класса:
```cpp
Base* ptr = new Derived();
ptr->show(); // Вызовет Derived::show()
```

Без `virtual` был бы вызван `Base::show()`. 

- **Полиморфизм:** Позволяет писать общий код, который работает с разными, но связанными типами данных (через базовый класс).
- **Динамическое связывание (Позднее связывание):** Определение, какая функция будет вызвана, происходит во время выполнения программы, а не на этапе компиляции.
- **Абстрактные классы:** Виртуальные функции, не имеющие реализации (например, `virtual void foo() = 0;`), делают класс абстрактным, не позволяя создавать его объекты, но обязывая наследников их реализовать (или сделать их чисто виртуальными

```cpp
class Animal {
public:
    virtual void makeSound() {  // Виртуальный метод
        std::cout << "Some sound\n";
    }
    
    virtual ~Animal() {}  // Виртуальный деструктор (обязателен!)
};

class Dog : public Animal {
public:
    void makeSound() override {  // Переопределение
        std::cout << "Woof!\n";
    }
};

int main() {
    Animal* animal = new Dog();
    animal->makeSound();  // Вызовет Dog::makeSound() - полиморфизм!
    delete animal;
}
```
