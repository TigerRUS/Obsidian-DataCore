Когда мы отлаживаем код, часто хочется быстро вывести значения переменных. 
Перепишем
```c++
std::cout << "x: " << x << std::endl;
```

```c++
#include <iostream>

#define LOG(var) std::cout << #var << " = " << (var) << std::endl;

int main() {
    int x = 42;
    double pi = 3.1415;

    LOG(x);   // x = 42
    LOG(pi);  // pi = 3.1415
}
```

`#var` превращает имя переменной в строку. А `(var)` - значение.

Такой макрос отлично заходит при написании алгоритмов, отладки функций, проверки значений — и при этом делает код аккуратнее.
Улучшение - можно сделать вывод в файл или добавить таймстемпы.
