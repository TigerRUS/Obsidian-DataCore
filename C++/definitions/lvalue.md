**lvalue** (от _locator value_) — это выражение, которое указывает на конкретную область памяти (объект), имеющую определенный адрес.

Если проще: *lvalue* — это то, что может стоять *слева* от оператора присваивания. У него есть имя, и оно *живет дольше, чем одна строка кода*.

Основные признаки lvalue:
1. Имеет *имя*.
2. Можно получить его *адрес* в памяти через оператор `&`.
3. Объект продолжает существовать после завершения выражения, в котором он был использован.

```cpp
int x = 10;      // 'x' — это lvalue. У него есть имя и адрес в памяти.
x = 20;          // Мы можем присвоить ему новое значение.

int* p = &x;     // Можно взять адрес lvalue.

std::string s = "Hello";
s[0] = 'h';      // Оператор [] для строки возвращает lvalue (ссылку на символ).
```

Сравнение с rvalue
В отличие от lvalue, **rvalue** — это временные значения, которые обычно не имеют имени и уничтожаются сразу после использования (например, числа `5`, `42` или результат сложения `x + 5`).

```cpp
int a = 5;
int b = 10;

int c = a + b;   // 'a' и 'b' — это lvalue.
                 // 'a + b' — это rvalue (временный результат, у которого нет имени).

// ПРИМЕР ОШИБКИ:
// 10 = a;       // Ошибка! 10 — это rvalue, ему нельзя ничего присвоить.
// & (a + b);    // Ошибка! Нельзя взять адрес у временного значения rvalue.
```

- **lvalue** — это «коробка» с именем, в которую можно что-то положить.
- **rvalue** — это «содержимое», которое мы переносим в коробку.
