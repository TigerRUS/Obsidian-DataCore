**Список инициализации членов (member initializer list)** в  — это механизм для установки начальных значений полей класса *до выполнения тела конструктора*.

Список начинается с двоеточия `:` после заголовка конструктора и идет до открывающей фигурной скобки:
```cpp
class MyClass {
    int x;
    int y;
public:
    // x инициализируется значением a, y — значением b
    MyClass(int a, int b) : x(a), y(b) {
        // Тело конструктора
    }
};
```

1. **Производительность:** Поля инициализируются напрямую. Если присваивать значения внутри `{ }`, объект сначала создается с дефолтным значением, а затем перезаписывается
2. **Единственный способ инициализировать:**
    - `const` поля
    - Ссылочные типы (`&`)
    - Поля базовых классов или члены-объекты, у которых нет конструктора по умолчанию
3. **Порядок:** Члены всегда инициализируются в порядке их **объявления в классе**, а не в порядке их написания в списке инициализации.
