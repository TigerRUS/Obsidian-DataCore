В C++ есть функция `std::exchange`, которая заменяет старое значение новым и возвращает старое, и оператор `swap`, который меняет значения двух переменных местами; оба находятся в (для `exchange`) или (для `swap`), позволяя эффективно управлять данными, а примеры показывают их использование с целыми числами, где `exchange(a, b)` присвоит `b` в `a` и вернет исходное значение `a`, а `swap(a, b)` просто поменяет их местами. 

**1. `std::exchange` (C++14)**  
Эта функция копирует новое значение в старое и возвращает старое значение.

 **Заголовочный файл**: `<utility>`.
```c++
#include <iostream>
#include <utility> // Для std::exchange

int main() {
    int old_value = 10;
    int new_value = 20;

    // Присваивает 20 в old_value и возвращает старое значение (10)
    int returned_value = std::exchange(old_value, new_value);

    std::cout << "Старое значение: " << returned_value << std::endl; // Выведет: 10
    std::cout << "Новое значение old_value: " << old_value << std::endl; // Выведет: 20
    std::cout << "Значение new_value: " << new_value << std::endl; // Выведет: 20

    return 0;
}
```

**Использование:** Полезно, когда нужно сохранить старое значение, одновременно обновляя переменную, например, в потоках или при работе с умными указателями. 

**2. `std::swap`**  
Просто меняет местами значения двух переменных.

**Заголовочный файл**: `<algorithm>` (или `<utility>`).
```c++
#include <iostream>
#include <algorithm> // Для std::swap

int main() {
    int a = 10;
    int b = 20;

    std::cout << "До обмена: a = " << a << ", b = " << b << std::endl; // Выведет: a = 10, b = 20

    std::swap(a, b); // Меняет местами значения a и b

    std::cout << "После обмена: a = " << a << ", b = " << b << std::endl; // Выведет: a = 20, b = 10

    return 0;
}
```
