*Entity-Component-System (ECS) «Сущность–компонент–система»* — это архитектурный шаблон программного обеспечения, используемый в разработке видеоигр для представления объектов игрового мира. ECS состоит из сущностей, составленных из компонентов данных, и систем, работающих с этими компонентами. ECS отдаёт приоритет композиции над наследованием.
